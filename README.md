# 🌾 UZHU AI
### AI-Powered Decision Intelligence Platform for Smart Agriculture

UZHU AI is an intelligent decision support platform that helps farmers make data-driven agricultural decisions using AI, historical crop data, weather forecasting, and government schemes.

---

## 🚀 Features

### 📊 Dashboard
- Live agricultural statistics
- Crop Explorer
- Production analytics
- State-wise crop insights
- BigQuery integration

### 📈 Analytics
- Production trends
- Yield analysis
- Crop comparison
- State-wise analytics

### 🧠 AI Decision Copilot
- Historical crop analysis
- Live weather intelligence
- 7-day weather forecast
- AI-powered farming recommendations
- Weather risk assessment
- Final farming decision

### 💬 Ask UZHU AI
- AI-powered agricultural chatbot
- Crop guidance
- Farming best practices
- Natural language interaction using Vertex AI

### 🏛 Government Schemes

- State-specific agricultural schemes

### 🌍 Community Impact

- Decision intelligence for communities

## 🎯 Problem Statement

Farmers often make decisions without access to real-time weather, historical agricultural data, AI recommendations, or government support information.

UZHU AI combines these sources into one intelligent platform to improve productivity, sustainability, and informed decision-making.
